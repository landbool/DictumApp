🌐 Dictum

Dictum — это продвинутая и удобная Студия Автоматизации, которая превращает ваш компьютер в интеллектуальную систему, управляемую удаленно. С помощью визуального конструктора макросов, интеграции с Telegram и умным домом Яндекса (Алиса), вы можете управлять вашим ПК из любой точки мира.

Проект сочетает в себе мощный локальный интерпретатор сценариев (эмуляция клавиатуры/мыши через нативный WinAPI, компьютерное зрение для поиска элементов на экране, озвучка через нейросети) и сетевой мост на базе протокола MQTT.

✨ Основные возможности

🎮 Визуальный редактор макросов:

Условия и логика: Проверка активности программ (if_active), запущенных процессов (if_proc), открытых сайтов (if_site) или значений переменных (if_var).

Циклы и ветвление: Поддержка циклов (loop), циклов с условиями (loop_while), случайных блоков (random_block) и прерываний.

Периферия: Нажатия комбинаций клавиш (с нативным обходом раскладки), управление курсором мыши, прокрутка, симуляция набора текста.

👁️ Компьютерное зрение (OpenCV): * Функция умного наведения: поиск графических шаблонов и кнопок на экране с регулируемой точностью для автоматического клика по ним.

🗣️ Озвучка Светлана Premium (Edge-TTS):

Локальная генерация и кэширование нейросетевых аудиофайлов для проговаривания статусов выполнения или озвучивания сценариев.

📱 Защищенный мост управления через Telegram (MQTT):

Шаг-за-шагом авторизация: Безопасное сопряжение конкретного ПК с аккаунтом в Telegram через одноразовые OTP-коды верификации.

Динамическая клавиатура: Кнопки пульта управления в Telegram автоматически синхронизируются с ПК и перестраиваются под ваш индивидуальный список макросов.

Интерактивный контроль: Запрос live-скриншотов со всех экранов ПК прямо в чат Telegram.

🎙️ Интеграция с Алисой (Домовёнок Кузя):

Мгновенное выполнение любых голосовых макросов через собственный MQTT-канал без задержек.

🎨 Премиальный Glassmorphism UI:

Современный адаптивный интерфейс на базе CustomTkinter с темной темой, красивыми кастомными тостами, плавными анимациями появления, заставкой при старте и встроенным терминалом логов.

🚀 Установка и запуск

Требования

Операционная система: Windows 10 / 11 (для корректной работы системных WinAPI хуков и позиционирования окон).

Установленный Python 3.10+.

1. Клонирование репозитория

git clone [https://github.com/ВАШ_ПРОФИЛЬ/dictum.git](https://github.com/ВАШ_ПРОФИЛЬ/dictum.git)
cd dictum


2. Установка зависимостей

Установите все необходимые библиотеки Python:

pip install -r requirements.txt


Основные зависимости: customtkinter, paho-mqtt, pyautogui, keyboard, psutil, opencv-python, numpy, pillow, pygame, edge-tts, requests, pygetwindow.

3. Настройка окружения (.env)

Создайте файл .env в корневой папке проекта со следующим содержимым:

TG_BOT_TOKEN=8674676096:AAFUYpw2GKfmP7q5-a4x2nOpJUmVjJ_n-iU  # Токен вашего Telegram-бота
MQTT_BROKER=broker.emqx.io                                    # Публичный или ваш личный MQTT-брокер


4. Развертывание Telegram-бота

Чтобы удаленное управление заработало, Telegram-бот должен быть запущен на сервере (или параллельно на ПК):

python bridge_tg.py


5. Запуск Студии Dictum

Запустите основное приложение Студии:

python main.py


🔒 Безопасность и сопряжение аккаунта

Запустите Студию main.py и перейдите в раздел Настройки.

Нажмите кнопку Шаг 1, чтобы перейти в бота @MyDictumBot и отправить ему ваш контакт (номер телефона).

В Студии введите этот же номер телефона и нажмите Шаг 2. Запросить код. Бот пришлет одноразовый 6-значный OTP-код.

Введите полученный код в поле шага 3 и нажмите Верифицировать.

Ваш ПК успешно привязан! В боте разблокируется клавиатура управления вашими макросами.
